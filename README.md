YouTube Clone – Interactive Video Streaming Interface

A responsive video streaming web application inspired by YouTube
, built using HTML, CSS, and JavaScript. This project recreates the core user interface and interactive experience of a modern video platform while focusing on frontend development, UI design, and DOM manipulation.

Features:
Responsive navigation bar and sidebar menu
Interactive hamburger menu animation
Real-time video search functionality
Category-based video filtering
Clickable video thumbnails linked to actual videos
Hover animations for video cards
Notification, voice search, and create button interactions
Modern UI inspired by YouTube
Tech Stack:
HTML5
CSS3
JavaScript (DOM manipulation & event handling)
What I Learned:
DOM manipulation
Event listeners
Interactive UI design
Responsive web development
Building real-world frontend projects

This is a beginner-to-intermediate frontend project demonstrating practical web development skills.
